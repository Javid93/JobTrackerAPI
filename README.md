# JobTrackerAPI

Production-ready ASP.NET Core Web API with JWT authentication, role-based authorization, Entity Framework Core, MySQL database, logging, Swagger/OpenAPI, and Azure deployment.

## About

JobTracker API is a backend system that allows users to manage and track job applications, including company name, position title, application status, applied date, and notes.

The API helps users keep track of:

- jobs they want to apply for
- jobs already applied to
- interview processes
- rejected applications
- job offers

## Technologies

- ASP.NET Core Web API
- Entity Framework Core
- MySQL
- Swagger / OpenAPI
- JWT Authentication
- REST API
- Azure Deployment

## Features

- Create job applications
- Retrieve all applications
- Prevent duplicate applications
- Database persistence with MySQL
- Swagger API documentation
- Entity Framework Core migrations
